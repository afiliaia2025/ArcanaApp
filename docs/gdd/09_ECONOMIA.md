## Modelo de Negocio

| Tier | Precio | Contenido |
|---|---|---|
| **Gratuito** | $0 | Ignis completo (Mates), práctica limitada (5/día) |
| **Familiar** | ~5€/mes | 4 asignaturas, práctica ilimitada, dashboard padres, eventos |
| **Centro Escolar** | Por alumno/año | Todo + dashboard profesor, Modo Aula, analítica, escáner IA |

### Principios Económicos

1. **El contenido educativo NUNCA se bloquea por dinero**. Solo se bloquean asignaturas extra, no capítulos.
2. **Sin compras in-app**. Solo suscripción.
3. **Sin anuncios. NUNCA.**
4. **Sin loot boxes ni mecánicas de azar.**

## Engagement y Retención

### Duración Real del Contenido

| Tipo | Sesiones | Tiempo |
|---|---|---|
| Capítulos narrativos | 39 | ~4.5h |
| Eventos estacionales | 12 | ~1.5h |
| Mini-bosses | 16 | ~1.5h |
| Bosses trimestrales | 3 | ~30 min |
| Boss final | 1 (5 fases) | ~20 min |
| Desafíos extra | 48 | ~2.5h |
| **TOTAL narrativo** | **~63** | **~12h** |
| Tareas del profesor | ~30 semanas | ~5h+ |
| Modo Práctica (IA) | ∞ | Sin límite |

### Mecanismos de Retención

| Mecanismo | Descripción |
|---|---|
| **Límite diario** | 2 caps nuevos/día. Distribuye 60 caps en 30 semanas |
| **Estrellas** | 3 por cap. El mapa las muestra. El niño quiere 3/3 |
| **Ejercicios rotativos** | Al repetir cap, preguntas diferentes |
| **Rachas** | Días consecutivos → recompensa creciente |
| **Eventos** | 4 mini-arcos estacionales por fecha |
| **Cosméticos** | Skins del bastón al completar reinos con 3★ |
| **Tablas de récords** | El niño compite consigo mismo |
| **Notificaciones** | Narrativas: *"Orión te espera..."* (máx 1/día, padres controlan) |
